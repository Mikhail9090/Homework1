How to execute this test profect:

1) Find Tests.java file
2) Find "gecDrPath" parameter and specify a path to the location of your Geckodriver
3) Find "userAgent" parameter and specify a desired mobile iOs user agent. Execute public void testMobileIphone() method to check Point 1.1 of homework
4) Find "desiredCar" parameter and type a desired car you would like to find. Execute public void testAvito() method to check Point 1.2 of homework
5) Find "screenPath" parameter and specify a desired path of a screenshot. Execute public void makeScreen() method to check Point 1.3 of homework
